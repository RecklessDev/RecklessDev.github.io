The idea is simple: matching 3 elements on the same type and level will create an element of the same type with a higher level. I struggled for about 45 minutes to reach a level higher than 3 and was a bit fun.

After I get the time to refactor the code a bit I will also publish it here; if it has enough interest maybe put in on the Asset Store for anyone to use.
